# 문제풀이 코드
```java
class Solution{
public int solution(int[] num_list,int n){
int answer=0;
for(int i=0;i<num_list.length;i++){
if(num_list[i]==n){
	answer=1;
    break;
}else{
	answer=0;
    }
}
return answer;
}
}
```
<hr>

# 문제 설명
1. num_list 배열의 길이만큼 반복되는 일반 for문을 사용해준다.
2. if문을 사용해서 배열 인덱스 번호에 n을 비교하는 조건문을 사용해 문제를 풀었다.<br>
3. n과 비교했을 때 n이 있다면 `answer=1`값을 넣어주고 break;문을 사용해서 n이 들어있다면 그때 if문을 빠져나오게 만든다.else문에는 n이 없는 경우인`answer=0`을 넣어주어 문제를 풀었다.
