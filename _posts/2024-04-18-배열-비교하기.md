# 문제 풀이 코드
```java
class Solution {
    public int solution(int[] arr1, int[] arr2) {
        int sum=0;
        int sum1=0;
        int answer = 0;
        if(arr1.length != arr2.length){
            if(arr1.length >arr2.length){
                answer =1;
            }else{
                answer = -1;
            }
        }else{
          for(int i=0;i<arr1.length;i++){
              sum+=arr1[i];
          }
            for(int i=0;i<arr2.length;i++){
                sum1+=arr2[i];
            }
            if(sum1>sum){
                answer=-1;
            }else if(sum>sum1){
                answer=1;
            }else{
                answer=0;
            }
        }
        
        
        
        return answer;
    }
}
```text
***
# ✨문제 풀이
1. arr1 배열의 원소들의 합을 구하는 sum변수,arr2배열의 원소들의 합을 구하는 sum1변수를 0으로 선언하고 초기화해준다.
2. if문을 사용해서 배열의 길이가 같지 않고,arr1배열의 길이가 더 클때 answer에 1을 넣어준다. else문을 사용하여 배열의 길이가 같지 않고 arr2배열의 길이가 더 크다면 answer에 -1을 담아준다.
3. else 문을 사용하여 배열의 길이가 같을 때 for문을 사용하여 sum변수에 arr1배열의 원소합을 담아주고 sum1변수에는 arr2배열의 원소합을 담아준다.
4. if문을 사용하여 sum1이 >sum보다 클때 answer변수에 -1을 담아주고, else if 문을 사용하여 sum변수가 더 클경우 answer 변수에 1을 담아주고, 마지막으로 else문을 사용하여 sum1과 sum이 같을 때 answer 변수에 0을 담아주어 문제에 맞게 해결하였다.
