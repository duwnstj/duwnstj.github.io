## 문제 설명
![](https://velog.velcdn.com/images/duwnstj12/post/76db0d68-7dcd-4fd0-b6d8-7c4e4e92568f/image.png)

## 첫번째 풀이 코드(실패)
```java
import java.util.*;
class Solution {
    public String solution(String s) {
        String answer = "";
       char[] charArray = s.toCharArray(); //1
        Arrays.sort(charArray);//2
        int index =0;
        for(int i = charArray.length-1;i>=0;i--){ //3
            
            charArray[index++] = charArray[i]; //4 
            
        }
        answer = String.valueOf(charArray); //5
        
        return answer; //6
    }
}
```
## 첫번째 문제 설명(실패)
- **주석으로 풀이**
1. 문자열 s를 t.CharArray()메서드를 통해 문자배열로 변환시킨다.
2. `Arrays.sort()`메서드를 사용해 charArray를 오름차순 정렬을 시켜준다.
3. 역순으로 저장하기 위해서 index 변수를 만들어 for문으로 역순으로 담기게 작성해봤다.
4. index0번에 index마지막번호에 해당하는 값이 담기고 그 다음 값이 담기고 결국 역순으로 변하게 된다(내 생각이었다).
5. answer에 charArray를 문자열로 변환시켜 저장시킨다. 
6. 그렇게 저장된 answer를 반환하면 문제가 해결될줄 알았다.

## 코드의 문제점
- 작성을 완료 하고 코드 테스트를 해봤더니 예상과 다른 결과가 나왔다. 
- 그 이유를 생각을해봤고 그림을 그려봤다. 
![](https://velog.velcdn.com/images/duwnstj12/post/e92be0a3-11c5-477d-9691-8fc12cb20ee8/image.png)
- 그림을 그려보니 A에 담기는 과정을 살펴보다보니까 내가 원하는 값은 
5,4,3,2,1 이렇게 역순으로 담기는 것이였는데, 결국 담기는 값은 5,4,3,4,5 가 담기게되는것이었다.
- **이전 반복문에서 인덱스 2번에 4번 값을 담았는데 4번 인덱스에도 4번값이 담기게 되니 이런 결과가 나오는 것이었다..**
- GPT를 통해 Collection에서 reverse()라는 메서드가 있다는걸 알게되었고 reverse()메서드를 사용해서 문제를 풀었다.

## 두번째 문제 풀이 코드(성공)
```java
import java.util.*;
class Solution {
    
    public String solution(String s) {
        char[] charArray = s.toCharArray(); // 1
        
        List<Character> charList = new ArrayList<>(); //2
        for(char b : charArray){ //3
            charList.add(b);
        }
        Collections.sort(charList); //4
        
        Collections.reverse(charList); //5
        
        StringBuilder answer = new StringBuilder(); //6
        for(char a : charList){ //7
            answer.append(a);
        }
        
        
        
        return answer.toString(); //8
    }
}
```
## 두번째 문제 설명(성공)
1. toCharArray() 메서드를 통해 s문자열을 배열로 만들어준다.
2. `ArrayList`를 사용하여 문자배열을 가변리스트로 변환한다.
3. 향상된 for문을 사용하여 charArray배열의 요소들을 charList 가변배열에 .add()메서드를 사용하여 넣어준다.
4. `Collections.sort()` 메서드는 컬렉션 리스트를 오름차순으로 정렬해준다. 즉 charArray를 인자로 넣어주면 charArray가 오름차순 정렬이된다.
5. `Collections.reverse()`메서드는 컬렉션 리스트를 역순으로 배열을 뒤집어준다. 즉 오름차순 정렬 후 reverse()메서드를 사용하면 내림차순으로 정렬된다.
6. StringBuilder 클래스를 이용하여 가변 문자열 `answer`를 만들어준다.
7. 향상된 for문을 이용하여 charList의 요소들을 answer가변 문자열에 StringBuilder클래스의 .append()메서드를 이용하여 저장시켜준다.
8. 가변 문자열인 answer를 .toString()메서드를 통해 문자열로 변환시켜 리턴해준다. 이렇게 문제를 해결했다.  

## 느낀점 
- 처음에는 왜 내가 짠 코드가 역순으로 저장이 안되는지 이해가 안갔었지만 그림을 그려보니 알게 되었다.
- `Collections` 클래스의 `reverse()` 메서드를 사용하면 내림차순을 쉽게 구현할 수 있다는 것을 알게 되었다.
- 앞으로는 이러한 문제를 만났을 때, `Collections` 클래스를 활용하는 방향으로 접근해야겠다.
