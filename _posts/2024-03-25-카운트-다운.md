# 문제 풀이 코드
{% highlight java %}
class Solution {
    public int[] solution(int start, int end_num) {
        int[] answer = new int[start-end_num+1];
        for(int i=start, j =0 ;i>=end_num;i--,j++){
            answer[j]+=i;
            
        }
        return answer;
    }
}
{% endhighlight %}
***
# 문제 풀이
1. 먼저 answer의 배열길이를 **start-end_num+1**(3까지 배열에 담겨야하기 때문에)로 선언해준다.
2. for문에 j와 i를 같이 선언해서 i의 값은 1씩줄고 j는 0부터 1씩 더해주는 식을 짜서 
`answer[j]+=i;`로 문제를 해결했다.
