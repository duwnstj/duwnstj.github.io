## 문제 풀이
```java
class Solution {
    public int solution(int n) {
        int answer = 0; //1
        String r="";  //2
        
        while(n!=0){ //3
            r += String.valueOf(n % 3);
            n = n/3;
        }
         // r = string 0021 
        
        for(int i=0;i<r.length();i++){ //4
            char change = r.charAt(r.length()-1-i); //5
            int result= Character.getNumericValue(change); //6
            answer += result * (int)Math.pow(3,i); //7
            }
        
        return answer;
    }
}
```
## 문제 설명
- 이 문제를 풀면서 10진법을 3진법으로 구하는 방법을 알게되었다.
- 10진법을 3진법으로 나타내는 방법은 이러하다
    - 숫자를 3으로 나누고 나머지를 기록한다.
    - 몫을 다시 3으로 나누고 나머지를 기록한다.
    - 기록된 나머지들을 역순으로 나열한다.
- 하지만 문제가 원하는건 3진법을 역순으로 나타내는것이기에 마지막 역순으로 나열하는 부분은 제외해도된다. 
- 주석으로 풀이 : 
1. 리턴받을 answer 변수를 선언해준다.
2. 나머지를 문자열로 받을 r 변수를 선언해준다.
3. while문을 사용하여 r변수에 n%3을 한값을 담는다. 한마디로 나머지값을 저장한다. 
    - 하지만 이 때 정수로 r 변수에 담게되면 각자리에 해당하는 나머지 값이 저장이 되야하는데 그게 아닌 나머지 값들이 전부 더해진 값이 저장이 되기 때문에 String.valueOf()메서드를 사용하여 문자열로 바꿔준다.
4. 역순의 3진법을 다시 10진법으로 만들기 위해서는 이러한 방법을 사용해야한다.
    - 각 자리 수를 확인한다.
    - 각 자리 수를 3의 제곱수와 곱한다.
    - 곱한 값을 모두 더한다.
**주석에 적힌 코드 설명은 문자열 길이만큼 for문을 돌린다.**
5. r의 각 자리수를 가져와야하는데 3의0승부터 n승까지 담기위해서 맨 뒷자리부터 가져와 change변수에 넣어준다.
6. result변수에 Character.getNumericValue()메서드를 사용해 '단일문자'의 값을 정수로 가져온다. 예를 들자면 '0'은 0으로 반환해준다
7. 각 자리 수를 뒷자리부터 가져와 `(int)Math.pow(3,i)`와 곱해주어 answer에 더해주고 그 값을 리턴 받으면 문제를 해결할 수 있다
    - 여기서 `Math.pow(3,i)`메서드를 사용하면 3의 i승의 값을 가져올 수 있고 Math.pow()메서드가 double타입으로 리턴하기 때문에 (int)를 사용하여 강제 형변환을 시켜준다.
