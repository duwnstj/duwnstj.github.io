## 문제 풀이
```java
class Solution {
    public int solution(int[][] sizes) {
        int maxwidth =0 ;
        int maxheight = 0 ;
        
        for(int i=0;i<sizes.length; i++){ //1
            int width = sizes[i][0];
            int height = sizes[i][1];
           
           // 세로가 가로보다 클 경우
            if(width < height){  //2
                int temp = width;
                width = height;
                height = temp;
            }
            // 가로의 제일 큰값 구하기
            if(width>maxwidth){ //3
                maxwidth = width;
            }
            //세로의 제일 큰 값 구하기
            if(height > maxheight){ //4 
                maxheight = height;
            }
            
        }
        return maxwidth * maxheight; //5
    }
}
```
### 문제 설명
- 처음에는 이중 for문을 써야하나 생각을 했지만 문제를 분석 한 후 결국 세로와 가로의 길이를 비교해서 세로가 가로보다 크다면 가로와 세로의 길이를 변환 , 가로의 큰 값, 세로의 큰 값을 곱하면 되겠다는 결론이 나오게 되었다.
- 따라서 for문을 사용하여 문제를 해결 했다.
1. 2차 배열의 행의 길이만큼 반복문을 돌려준다. width에는 가로의 값을 , height에는 세로의 값을 담아준다. 
2. 만약 세로가 가로보다 클 때 temp 변수를 사용하여 서로 값을 바꿔준다. 
    - 만약 a=10와 b=5라는 변수가 있다고 할 때 a=b,b=a 이렇게 한다고 하면
    **결국 a와 b의 결과가 5로 같은 경우가 나오게 될것이다.**
    - 왜냐하면 먼저 b의 값이 a에 담겼기 때문에 a가 b의 값을 갖게 되고 b는 그 a의 값을 가져오기 때문이다.
    - 따라서 temp변수를 사용하여 가로의 값을 먼저 temp에 저장하고 width에 세로의 값을 저장한 후 height에는 가로의 값을 저장하는것이다. 이로써 가로와 세로의 값을 바꿔줄 수 있다.
3. 만약 width가 maxwidth보다 클 때 maxwidth에 가로의 최대값을 저장해준다.
4. 3번과 같은 로직이다. 
5. 가로의 최대값, 세로의 최대값을 리턴해주면 문제를 해결 할 수 있다.

## 다른 사람이 푼 방법
```java

class Solution {
    public int solution(int[][] sizes) {
        int length = 0, height = 0;
        for (int[] card : sizes) { //1
            length = Math.max(length, Math.max(card[0], card[1])); //2
            height = Math.max(height, Math.min(card[0], card[1])); //3
        }
        int answer = length * height; //4
        return answer;
    }
}
```
- 이 사람은 Math.max()함수를 이용하여 문제를 풀었다.
1. 2차 배열 sizes를 1차배열 card로 향상된 for문을 이용하여 반복한다. 
2. length에 가로의 최대값을 저장한다. 
    - card[0]은 가로 , card[1]은 세로를 나타내는데 Math.max()를 사용하여 그 값들을 서로 비교 후 더 큰 값을 반환한다. 
    - 그 값을 한번더 length와 비교하여 그 중 가장 큰 값 즉 가로의 최댓값을 length에 저장하는 것이다.
3. height에 세로의 최대값을 저장한다.
    - Math.min()메서드를 사용하면 첫번째 인자와 두번째 인자를 비교해 더 작은 값을 반환해준다.
    - Math.min()메서드를 사용하여 가로와 세로의 길이를 비교 후 작은 값이 세로길이이기 때문에 그 값 중 최대값을 height에 저장해준다. 
    - 즉 세로의 최대값이 height에 저장이 되게 된다. 
4. 마지막으로 length와 height의 값을 곱해 answer에 저장한 후 반환 해주면 문제를 해결 할 수 있다.
