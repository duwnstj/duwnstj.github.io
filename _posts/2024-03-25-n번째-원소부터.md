# 📃문제 풀이 코드 
```
import java.util.Arrays;
class Solution {
    public int[] solution(int[] num_list, int n) {
        
        int answer[]= Arrays.copyOfRange(num_list , n-1 , num_list.length);
        
        return answer;
    }
}
```
***
# ⬛문제 풀이 
1.`Arrays.copyOfRange`함수를 사용해서 문제를 풀었다.
- 사용법 : `Arrays.copyOfRange(원본배열 , from , to)`으로 사용한다.
따라서 여기서는 원본 배열에 num_list를 넣어주고 `from에는 n-1`을 넣어주는 이유는 
인덱스 번호가 0부터 시작하기 때문이다. 그리고 to에는 num_list.length를 넣어준다.
이때 num_list.length만써주는 이유는 `Arrays.copyOfRange()`함수에서 to는 포함하지 않기 때문이다.
