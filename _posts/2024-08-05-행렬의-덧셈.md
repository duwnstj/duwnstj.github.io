## 문제 풀이 코드 
```java
class Solution {
    public int[][] solution(int[][] arr1, int[][] arr2) {
      int  lows = arr1.length; //1
       int cols = arr1[0].length; //2
        int[][] answer = new int[lows][cols]; //3
        for(int i=0;i<lows;i++){ //4
            for(int j=0;j<cols;j++){ 
                answer[i][j] = arr1[i][j]+arr2[i][j]; //5
            }
        }
        return answer;
    }
}
```
## 문제 풀이 설명
- **주석으로 풀이**
1. 행의 길이를 lows 변수에 저장한다.
2. 열의 길이를 cols 변수에 저장한다. `arr1[0].length`를 사용하면 열의 길이를 구할 수 있다.
    - 2번 예시 :`arr[0][0],arr[0][1],arr[0][2]` 이런 2차원 배열이 있다면 행의 길이는 `1`, 열의 길이는 `3`이 나오게 된다.
3. 정해준 lows 변수와 cols 변수를 answer의 길이로 배열을 새로 만들어준다.
4. 2중 for문을 사용하여 바깥 for문은 행의 길이만큼 돌게 만들어주고 ,안쪽 for문은 열의 길이만큼 돌수있게 for문을 사용한다.
5. answer 배열에 arr의 각 요소들을 더해 저장시키면 문제를 해결 할 수 있다.
- 이 문제는 arr1과 arr2의 행과 열의 크기가 같기 때문에 arr1의 길이를 통해서 문제를 해결 할 수 있다
