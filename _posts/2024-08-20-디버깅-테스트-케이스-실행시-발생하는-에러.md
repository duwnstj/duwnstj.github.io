# 오늘 겪은 트러블 슈팅
- 강의를 따라하면서 Test 코드를 실행 시켰는데 이러한 에러가 났다.
![](https://velog.velcdn.com/images/duwnstj12/post/6ec50f77-1b43-45ef-a6b0-0a0cd6675726/image.png)
- 어떤 에러인지 감이 안잡혀 구글링을 통해서 알게되었다. 
- 이 에러가 뜨는 이유는 JUNIT5 설정할 때 나는 오류라고 한다. 따라서 아래와 같이 디버깅을 하였다.

## 해결 방법
1.[File]-[settings] - [Build,Execution,Deployment] - [Gradle]을 들어간다.
2. Gradle Projects에서 변경할 프로젝트를 정해준다.
3. Build and run using,Run tests using을 `InteliJ IDEA`로 변경해준다.
![](https://velog.velcdn.com/images/duwnstj12/post/2b5e24fd-bd39-43ee-908b-4b4f1acc0cd7/image.png)

변경 후 다시 실행 해보니 문제 없이 실행되었다.
