# 문제 설명
![](https://velog.velcdn.com/images/duwnstj12/post/b1cd5f09-96e8-4838-8262-090ba7099327/image.png)

## 문제 풀이 코드
```java
class Solution {
    public String solution(int n) {
        String answer = ""; 
        for(int i=0;i<n ;i++){ // 1
            if(i % 2 ==0){ //2
                answer+="수";
            }else{ //3
                answer+="박";
            }
        }
        return answer;
    }
}

```

## 문제 풀이 설명
- **주석으로 풀이**
1. for문으로 n만큼 반복해준다.
2. if문을 사용하여 i가 짝수라면 answer에 "수"문자를 더해준다.
3. else문을 사용하여 i가 홀수라면 answer에 "박"문자를 더해준다.
4. 이렇게 answer를 리턴 받으면 n만큼 돌면서 원하는 문자열을 반환받을 수 있다.
