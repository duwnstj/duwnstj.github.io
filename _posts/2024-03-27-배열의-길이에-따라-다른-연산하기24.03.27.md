# 문제 실행 코드
```java
class Solution {
    public int[] solution(int[] arr, int n) {
        int[] answer = new int[arr.length];
        for(int i=0;i<arr.length;i++){
            if(arr.length % 2 !=0){
                if(i %2 ==0){
                answer[i] +=arr[i]+n; 
                    }else{
                    answer[i]=arr[i];
                }
            }else{
                if(i % 2 !=0){
                    answer[i]+=arr[i]+n;
                }else{
                    answer[i]=arr[i];
                }
            }
        }
        return answer;
        }
        
    }
```
***
# 문제 풀이
1. `answer의 배열길이`를 `arr의 배열길이`로 선언해준다.
2. for문 안에 arr배열의 길이가 홀수일때 실행되는 if문을 만들어주고 else문에는 arr배열의 길이가 짝수일때 실행되는 때를 작성해준다. arr배열의 길이가 홀수이면서 인덱스 번호가 짝수인 경우에는 `answer[i] += arr[i]+n;`을 해주고,arr배열의 길이가 홀수이면서 인덱스번호가 홀수인 경우에는 `answer[i]=arr[i]`를 해준다. 이렇게 해주면 인덱스번호가 홀수일때는 n을 더해주고 인덱스 번호가 짝수일때는 해당인덱스 번호에 해당되는값을 받아올수가있다. 
3. arr배열 길이가 짝수일때도 위를 응용하여 코드를 짜서 문제를 해결했다.
