## 문제 풀이 
```java
class Solution {
    public String solution(String s) {
        // 각 단어는 하나 이상의 공백 문자로 구분 되어짐
        // 각 단어의 짝수번째 알파벳->대문자 , 각 단어의 홀수번째 알파벳 -> 소문자
        // 문자열 전체의 짝/ 홀수 인덱스가 아니라 단어(공백을 기준)별로 짝/홀수 인덱스를 판단
        // 첫번째 글자는 0번째 인덱스로 짝수번째 알파벳으로 처리
        String answer = "";
        String[] result = s.split(""); //1
        
        int idx =0; //2
        for(int i=0;i<result.length;i++){ //3
                
            if(result[i].equals(" ")){ //4
                idx = 0;
            }else if(idx % 2 ==0){ //5
                result[i] = result[i].toUpperCase();
                idx++;
            }else if(idx % 2 != 0){ //6
                result[i] = result[i].toLowerCase();
                idx++;
            }
            answer += result[i] ; 
        }
        return answer;
    }
}
```
## 문제 설명
1. split()메서드를 사용하여 각 자리의 문자를 배열로 반환시켜서 result 변수에 담아준다.
2. 인덱스를 담을 변수 idx를 0으로 선언해준다.
3. for문을 통해 result 변수만큼 반복해준다.
4. result의 요소 중 만약 공백이라면 인덱스를 다시 0으로 초기화 해준다. 공백을 만나면 다시 인덱스를 0으로 초기화시켜 다시 검사한다. 
5. 인덱스가 만약 짝수라면 result 각 요소를 대문자로 변환시킨다. 그리고 인덱스에 ++을 시켜준다. 인덱스를 기준으로 대문자,소문자로 나누기 위해 이렇게 작성했다.
6. 인덱스가 만약 홀수라면 소문자로 변환 시킨다. 이때도 인덱스에 ++을 해준다.
7. answer에 각 result 요소들을 더해준다. 이렇게 문제를 해결했다.
