## 문제 풀이 코드 
```sql
-- 코드를 입력하세요
SELECT animal_id,name --1
from animal_ins --2
where intake_condition != 'Aged' --3
order by Animal_id asc ; --4
```
### 문제 설명
1. animal_id와 name을 조회한다.
2. animal_ins 테이블에서 조회한다.
3. intake_condition이 Aged가 아닐때 검색한다. 젊은 동물이라는것이 InTAKE_CONDITION이 Aged가 아닐 때이기 때문이다.
4. Animal_id를 기준으로 오름차순 정렬한다. 여기서 asc는 생략이 가능하다.
