> [가운데 글자 가져오기](https://school.programmers.co.kr/learn/courses/30/lessons/12903)

# 문제 설명
![](https://velog.velcdn.com/images/duwnstj12/post/cd27241e-1224-4a23-9bce-6a8c3aab74a4/image.png)

## 첫번째 문제 풀이 코드 
{% highlight java %}
class Solution {
    public String solution(String s) {
        String answer = "";
        for(int i=0;i<s.length();i++){ // 1
            if(s.length()% 2 == 1){//2
               answer=s.substring(s.length()/2,s.length()/2+1); //3
            }else{ //4
                answer = s.substring(s.length()/2-1 ,  s.length()/2+1); //5
            }
        }
        return answer;
    }
}
{% endhighlight %}
## 문제 풀이 첫번째 설명(성공)
- **주석으로 풀이**
1. for문을 사용하여 s의 문자열 길이만큼 반복한다.
2. if 문을 사용하여 s의 문자열 길이가 홀수일때를 조건으로 넣는다.
3. answer에 substring을 사용하여 가운데 문자를 추출하고 저장한다. **substring(a,b)메서드를 사용하면 문자열의 첫번째 인자부터 두번째 인자 미만 까지 해당하는 문자열을 반환해준다.** 
4. else 문을 사용하여 짝수일 경우도 구해준다.
5. answer에 substring을 사용하여 가운데 문자 두개를 반환해준다. 이렇게 해서 문제를 해결했다.

- 문제를 풀긴하였지만 이렇게 보니 너무 가독성이 떨어지는것같아서 가운데 문자를 구하기 위해 쓴 `s.length()/2`를 변수로 정해서 다시 풀어봤다.
- 또 for문없이 substring만 사용하여도 문제를 해결 할 수 있을 것같아 다시 문제를 풀었다.

## 두번째 문제 풀이 코드
{% highlight java %}
class Solution {
    public String solution(String s) {
        String answer = "";
            int num1 = s.length()/2;
            if(s.length()% 2 == 1){
               answer = s.substring(num1,num1+1);
            }else{
                answer = s.substring(num1-1 , num1+1);
            }
        
        return answer;
    }
}
{% endhighlight %}
## 두번째 문제 풀이 설명

- 첫번째와 문제 풀이 방법은 동일하지만 substring만을 사용하여도 가운데 문자열을 반환해주기때문에 for문을 제거한다. 
- num1이라는 변수에 `s.length()/2`를 한 값을 저장해서 문제를 푸니 코드의 가독성이 훨씬 좋아졌다.
- 이렇게 문제를 해결하고 나니 코드의 가독성이 훨씬 좋아졌고 불필요한 for문을 없애주어 코드의 성능이 좋아졌다.
