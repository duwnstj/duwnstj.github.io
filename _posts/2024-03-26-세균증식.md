# 문제풀이 코드 1
```java
class Solution {
    public int solution(int n, int t) {
        int answer = 0;
        for(int i=1;i<=t;i++){
           n *= 2;
            
        }
        
         return n;
    }
   
}
```
***
# 문제 풀이 1
1.일반 for문을 사용해 내가 설정한 t(시간)만큼 반복문이 실행될 수 있게 조건을 정해준다.
2. `n*=2;` `n*2`를 한값이 n에 담길 수있게 해준다. 이렇게 해주어야 `n값에 *2` 해준값이 담기고 그 값에 다시 *2를 해줄 수 있게 된다. n을 리턴 받으면 문제를 해결할 수있다.
***
# 문제 풀이 코드 2
```java
class Solution {
    public int solution(int n, int t) {
        int answer = 0;
        answer=n*(int)Math.pow(2,t);
        
        return answer;
        
        
    }
}
```
***
# 문제 풀이 
1. `math.pow()`함수를 사용해서 문제를 풀었다.
2. `math.pow()` 사용법 :`Math.pow(2,t)`라고 한다면 2를 t번 거듭제곱한것과 같은의미이다. 하지만 이때 주의 점이 있는데 double형식으로 반환을 해주기 때문에 앞에 (int)를 붙여서 강제 형변환을 해주고 나서 n과 다시 곱해준다.return 값으로 answer를 리턴 받으면 문제를 해결 할 수 있다.
