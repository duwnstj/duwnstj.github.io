## 문제 풀이 
```java
class Solution {
    public int solution(int[] number) {
        //number중 3명의 값들을 더해서 0이되는 경우의 수  가지 수 구하는 문제
        int answer = 0;
        for(int i=0;i<number.length;i++){ //1
            for(int j=i+1;j<number.length;j++){ //2
                for(int k=j+1;k<number.length;k++){ //3
                    if(number[i]+number[j]+number[k]==0){
                        answer++;
                    }
                }
            }
        }
        
        return answer;
    }
}
```
## 문제 풀이 설명

- 이 문제는 해석을 해보니 number 중 3명의 값들을 더해서 0이 되는 경우의 수 가지수를 구하는 문제였다.
- 3중 for문을 이용하여 문제를 해결했다.
1. 3가지 숫자의 값들을 더하기 위해 0번째 인덱스부터 number의 길이 미만까지 반복해준다.
2. 3가지 숫자의 값들을 더하기 위해 1번 인덱스 부터 number의 길이 미만까지 반복해준다.
3. 3가지 숫자의 값들을 더하기 위해 2번 인덱스 부터 number의 길이 미만까지 반복해준다.
- if문을 사용하여 3가지 숫자를 더했을 때 0이 된다면 answer++을 시켜 answer를 반환한다.
