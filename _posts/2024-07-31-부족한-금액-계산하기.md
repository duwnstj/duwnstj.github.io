## 🎶 문제 풀이 코드
```java
class Solution {
    public long solution(int price, int money, int count) {
    
        long result =0;
        for(int i=1;i<=count ; i++){// 1
            result += price * i; // 2
            
        }
        if(result>money){ // 3
             return result - money;
             
            }

        return 0;
    }
}
```
## 문제 풀이 설명
1. for문을 사용하여 `1`부터 `count`까지 반복하면서 각 놀이기구 탑승 비용을 누적한다.
2. 총 비뵹이 보유한 돈보다 클 때 부족한 금액을 반환한다.
3. 부족하지 않을경우 `0`을 반환한다.

## 다른 사람이 푼 코드
```java
class Solution {
    public long solution(int price, int money, int count) {
        long totalCost = price * (long)count * (count + 1) / 2; //1
        long shortage = totalCost - money; //2
        return shortage > 0 ? shortage : 0; //3
    }
}
```
## 문제 풀이 설명
1.등차수열의 합의 공식을 이용하여 놀이기구를 타기위한 총 금액을 구한다.
2. shortage 변수에 부족한 비용을 담는다.
3. 삼항 연산자를 사용하여 shortage가 양수일 때 shortage를 반환하고 , shortage가 양수가 아닌 경우 0을 반환한다.

## 등차수열의 합 공식
등차 수열의 합을 구하는 공식은 다음과 같다.
$S_n = \frac{n}{2} \times (a + l)$

- $n$: 항의 개수
- $a$: 첫 번째 항
- $l$: 마지막 항

## 공식 설명
- **정방향과 역방향 수열 작성** : 
등차 수열을 정방향과 역방향으로 작성하여 쌍의 합을 구한다.
- **각 쌍의 합** : 
 각 쌍의 합은 $a+l$이다. 
- **쌍의 개수** : 
 쌍의 개수는 $$\frac{n}{2}$$이다.
 
 ## 예제
 수열이 $1,2,3,…,10$인 경우 :
 - 첫 번째 항 ($a$): 1
- 마지막 항 ($l$): 10
- 항의 개수 ($n$): 10

## 느낀점
- 일반 for문을 사용한 방식과 등차수열 공식을 활용한 방식의 차이를 이해하게 되었다.
- 등차수열의 수학적 개념과 그 공식을 문제에 적용하는 방법을 배웠다. 아직 많이 미숙하다 .
- 알고리즘 문제를 풀면서 수학공식을 이해하는것이 중요하다는 것을 느꼈다.
