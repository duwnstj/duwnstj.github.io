# 문제 설명

## 첫번째 문제 풀이 코드
```text
class Solution {
    public int[] solution(int[] arr) {
        
        int[] answer = new int[arr.length-1]; //1
        int min = arr[0]; //2
        for(int i=0;i<arr.length;i++){ //3
            min = Math.min(min,arr[i]);
        }
        if(arr.length==1){ //4
            return new int[]{-1};
        }
        for(int i=0;i<arr.length;i++){ //5
           if(min==arr[i]){
               continue;
           }else{ //6
               answer[i] = arr[i];
           }
        }
        return answer;
    }
}
```text
### 문제 풀이 첫번째 설명(실패)
- **주석으로 풀이**
1. 제일 작은 수는 배열에서 제거하기 때문에 arr의 배열길이에서 -1을 해준 배열 크기로 answer 배열을 만든다.
2. `min` 변수에 `arr[0]`을 저장해준다.
3. `for문`을 사용하여 배열의 크기만큼 반복해준다.<br> min 변수에 `Math.min(a , b)`메서드를 사용하였다. Math.min(a,b)는 a와 b를 비교하여 작은 수를 리턴해준다.
4. 만약 arr배열의 길이가 1이라면 이때는 반환 해줄 값이 없기때문에 -1을 리턴해준다. 
5. for문을 사용하여 배열의 길이만큼 반복해주고, if문을 사용하여 만약 min과 arr[i]가 같을 때 그 값이 제일 작은 값이니 continue를 사용하여 제외 시킨다.
6. else 문을 사용하여 answer 배열에 arr의 배열의 모든 요소를 담아주어 문제를 풀었다.

- 하지만 i를 넣으니 문제 테스트에서는 잘 넘어갔지만 문제 제출에서 런타임 오류가 떳다. 
- **이유가 뭘까 생각을 해보니 처음에 answer배열을 arr의 길이보다 -1로 크기를 선언을 했는데 아래 for문에서는 answer 배열이 arr의 배열만큼 반복하기때문에 이런 오류가 나는것이었다.** 
- 따라서 아래와 같이 index 변수를 선언하여 문제를 해결했다.



 ## 두번째 문제 풀이 코드

```java
class Solution {
    public int[] solution(int[] arr) {
        
        int[] answer = new int[arr.length-1]; //1
        int min = arr[0]; //2
        for(int i=0;i<arr.length;i++){
            min = Math.min(min,arr[i]); //3
        }
        if(arr.length==1){ //4
            return new int[]{-1};
        }
        int count =0; //5
        for(int i=0;i<arr.length;i++){ //6
           if(min==arr[i]){
               continue;
           }else{ // 7
               answer[count++] = arr[i];
           }
        }
        return answer;
    }
}
```text
### 문제 풀이 두번째 설명(성공)
- **주석으로 풀이**
1. `arr.length-1`로 answer 배열의 크기를 선언한다.
2. min 변수에 `arr[0]`번 인덱스 값을 할당한다.
3. `for문`을 사용하여 배열의 크기만큼 반복해준다.<br> min 변수에 `Math.min(a , b)`메서드를 사용하였다. Math.min(a,b)는 a와 b를 비교하여 작은 수를 리턴해준다.
4. 만약 arr배열의 길이가 1이라면 이때는 반환 해줄 값이 없기때문에 -1을 리턴해준다. 
5. index 변수를 0으로 선언해준다.
6. for문을 사용하여 배열의 길이만큼 반복해주고, if문을 사용하여 만약 min과 arr[i]가 같을 때 그 값이 제일 작은 값이니 continue를 사용하여 제외 시킨다.
7. else 문을 사용하여 answer 배열에 arr의 배열의 모든 요소를 담아주어 문제를 풀었다.<br>이 때 카운트 변수에 증감연산자를 사용하여 arr의 모든 요소가 answer 배열에 저장될 수 있도록 문제를 해결했다.
