## 첫번째 문제 풀이(성공)
{% highlight java %}
import java.util.Scanner;

class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt(); //1
        int b = sc.nextInt(); //2
        
        for(int i=0;i<b;i++){ //3
            for(int j=0;j<a;j++){ //4
                System.out.print("*");
            }
            System.out.println();
        }

        
    }
}
{% endhighlight %}
### 문제 설명
- 주석으로 풀이
1. a를 입력 받은 정수로 저장해준다.
2. b를 입력 받은 정수로 저장해준다.
3. 이중 for문을 사용해 *이 a가 가로의 길이 , b가 세로의 길이로 나오게 만들어준다.
4. 이렇게 하면 문제를 해결 할 수 있다.

## 두번째 문제 풀이(성공)
{% highlight java %}
import java.util.Scanner;

public class Solution {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt(); // 1
        int b = sc.nextInt();  // 2
        StringBuilder sb = new StringBuilder(); //3
        for(int i=0; i<a; i++){ //4
            sb.append("*");
        } 
        for(int i=0; i<b; i++){ //5
            System.out.println(sb.toString());
        }
    }
}
{% endhighlight %}
### 문제 설명
- 이번 방법은 다른 사람은 어떻게 풀이 했을까하면서 블로그를 보고 참고해서 풀어보았다.
- 이중 for문을 사용하지 않고 `StringBuilder`클래스를 이용하여 문제를 풀었다.
- **주석으로 풀이** : 
1. 입력받은 정수를 a에 저장한다.
2. 입력받은 정수를 b에 저장한다.
3. 가변 문자열 sb를 만들어준다.
4. `append()`메서드를 사용하여 *을 a 즉 가로의 길이 만큼 sb에 추가해준다.
5. b 즉 세로의 길이만큼 sb에 추가한 별을 `.toString()`메서드를 사용하여 문자열로 변환해 출력한다.
- 이 방법은 문자열 sb를 한번만 생성하면 되기 때문에 위의 방법보다 메모리적 사용측면에서 효과적일 수 있다.
