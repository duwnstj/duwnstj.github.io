# 문제 풀이 코드
```java

class Solution {
    public int solution(int[] num_list) {
        int answer = 0;
        String a="";
        String b="";
        int num1=0;
        int num2=0;
       for(int i=0;i<num_list.length;i++){
           if(num_list[i] %2==1){
               a += ""+num_list[i];
              num1= Integer.parseInt(a);
           }else if(num_list[i] %2==0){
                b+=""+num_list[i];
               num2= Integer.parseInt(b);
           }
           
       }
        answer = num1+num2;
        return answer;
        
    }
}
```
***
# 😁문제 풀이 
1. 정수들이 이어붙은 값이 나와야하기때문에 정수를 문자열로 바꿔야한다. 그렇기 때문에 a변수에 인덱스 해당 값이 홀수라면 a변수에 담기게 코드를 짰고 b변수에는 인덱스 해당 값이 짝수라면 b변수에 담기게 코드를 짰다. 그리고 이 값을 더해준 값을 리턴 받아야하기때문에 `Integer.parseInt`를 이용해서 정수로 다시 바꿔주고 그 값들을 더해줌으로써 문제를 해결했다.
