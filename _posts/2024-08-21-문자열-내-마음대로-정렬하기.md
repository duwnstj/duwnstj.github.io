## 문제 풀이 코드
```java
import java.util.*;
class Solution {
    public String[] solution(String[] strings, int n) {
        String[] answer = {};
        
        Arrays.sort(strings,Comparator.comparing((String s) -> s.charAt(n))// 1
            .thenComparing(s->s)); //2
        return strings;
    }
}
```text
## 코드 설명
1. **n번째 문자 기준 정렬** :  
`Array.sort()` 메서드를 사용하여 문자열 배열을 정렬한다.
여기서 `Comparator.comparing()` 메서드를 사용하여 각 문자열의 `n`번째 문자를 기준으로 정렬한다.
`s.charAt(n)`은 주어진 문자열 `s`의 `n`번째 문자를 반환하며 , 이 문자들이 오름차순으로 정렬된다.
예를 들어 `["sun","bed","car"]` 배열에서 `n= 1`이라면 `"u","e","a"` 순서로 정렬된다.

2. **사전순 정렬** :
만약 `n`번째 문자가 동일한 경우, `thenComparing()`메서드를 사용하여 문자열 전체를 사전순(알파벳순)으로 정렬한다.
예를 들어, 배열 `["abcd","abce","cde"]`에서 `n=2`라면 `c,c,e`로 정렬되는데 두번째 문자가 같은 경우 `abcd`와 `abce`는 문자열 전체를 기준으로 사전순으로 비교된다. 따라서 `abcd`가 `abce`보다 앞서게된다.
3. 정렬된 `strings` 배열을 그대로 반환하여 문제를 해결 할 수 있다.
