# ✨문제 풀이 코드
```java
class Solution {
    public int solution(int a, int b) {
        int answer = 0;
        String c =""+a+""+b;
        int d = Integer.parseInt(c);
        
        if(d>2*a*b){
            answer=d;
        }else if(d<2*a*b){
            answer=2*a*b;
        }else{
            answer=d;
        }
        return answer;
    }
}
```
***
# 😁문제 풀이
1. 정수를 문자로 변환시켜서 두정수를 붙인 값을 문자열 변수 C에 저장한다.
2. c의 값을 정수로 변환시켜서 d 변수에 저장시킨다
3. if~else문을 사용해서 d가 클때는 answer에 d를 반환 시키고 d가 작을때는 
`2*a*b`를 반환시키고 만약 같을때는 d를 반환하게 코드를 짜서 문제를 해결했다.
