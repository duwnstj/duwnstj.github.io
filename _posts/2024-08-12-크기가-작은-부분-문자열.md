## 문제 풀이 
```java
class Solution {
    public int solution(String t, String p) {
        int answer = 0;
        int len = p.length(); //1
        Long a = Long.parseLong(p); //2
        
        for(int i=0;i<t.length()-len+1;i++){ //3
            Long diff = Long.parseLong(t.substring(i,i+len));//4
            
            if(diff<= a){ //5
                answer++;
            }
        }
        
        return answer;
    }
}
```
### 문제 설명
- **주석으로 풀이 **:
1. len 변수에 p의 문자열 길이를 저장한다.
2. p를 Long 타입으로 변환한다. 처음에 `int`타입을 사용했더니 런타임 에러가 났다. 만약 t의 문자열 길이와 p의 문자열 길이가 int 범위보다 큰 경우가 있을 때에 에러가 나는것이다.
    - 따라서 Long 타입으로 변환해주었다.
3. 부분 문자열을 구하기 위해서 `t.length()-len+1`만큼 for문을 반복시켜줬다.
    - 예를 들어 t.length:8 , len = 3 이라면 총 6번을 반복을 해주어야한다.
4. .substring()메서드로 각 부분 문자열을 추출해서 비교하기 위해 Long타입으로 변환 시켜준다.
    - 여기서 `substring(i,i+len)`을 해주면 처음 반복문에는 0번인덱스부터 문자열길이까지 추출, 두번째 반복문에서는 1번 인덱스부터 문자열길이까지 추출 ... 이런식으로 모든 부분 문자열들을 추출 할 수 있다.
5. diff가 a보다 작거나 같을 때 answer를 ++를 해준다.
