# 문제 풀이 코드
{% highlight java %}
import java.util.Arrays;
class Solution {
    public int[] solution(int[] num_list) {
        int[] answer = new int[num_list.length];
         Arrays.sort(num_list);
    
        answer = Arrays.copyOfRange(num_list,0,5);
        
        return answer;
    }
}
{% endhighlight %}
***
# ✨ 문제 풀이
1. `Arrays.sort()`메서드를 사용해서 배열을 오름차순 정렬을 해준다.
2. `Arrays.copyOfRange()`메서드를 사용해서 num_list 배열의 인덱스0~4까지 잘라서 answer배열에 넣어주어 문제를 해결했다. Arrays클래스에 속한 메서드를 사용하려면`import java.util.Arrays;`를 꼭 임포트해주어야한다.
