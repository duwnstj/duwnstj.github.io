## 문제 풀이
```java
class Solution {
    public int[] solution(int n, int m) {
        int[] answer = new int [2]; //1
        int result = n*m;  //2
         
        int r=0;  //3
        while(m != 0){ //4
           r =n % m ; 
            n =m; 
            m =r; 
        }
        answer[0] =n; //5
        answer[1] = result/answer[0]; //6
        
       
        
        return answer;
    }
}
```
- 주석으로 풀이:
- 최대공약수를 구하기위해 매우 쉬운 공식이 있는데 이 공식이 바로 유클리드 호제법이다.
- 유클리드 호제법 공식은 이러하다. 
    - "2개의 자연수 a,b(a>b)에대해서 a를 b로 나눈 나머지가 r일때, a와 b의 최대공약수는 b와 r의 최대공약수와 같다."
1. 최대공약수와 최소 공배수가 담길 answer 배열 생성해준다.
2. 2번은 최소 공배수를 구하기 위해 저장할 변수를 선언해준것이다.
3. 나머지를 담을 변수를 선언해준다.
4. while문을 사용하여 m이 0이 아닐때 반복문을 돌려준다.
    - 4번에 관한 설명을 유클리드 호제법 공식을 이용해 설명해보겠다. 
    - 만약 n이 45이고 m이 5라고 할 때 `45 % 5 = r`; 여기서 r에 나머지 값이 담기고 n에는 m값(5)를 , m값에는 0이 나오기때문에 여기서 반복문을 나오게 된다.
    - 결국 최대 공약수는 n(==5)을 리턴하게 되는 것이다.
5. answer[0]에 n을 저장한다. 즉 최대공약수를 저장하게된다.
6. 최소 공배수를 구하는 성질은 `두 수 a와 b의 최소공배수는 a와 b의 곱을 a와 b의 최대공약수를 나눈것과 같다.` 이러한 성질이 있다.
    - 따라서 이를 활용하여 2번에서 선언한 result 변수를 최대공약수로 나눠 최소 공배수를 구해주고 그 값을 answer[1]번에 담았다.
7. 이렇게 풀면 문제를 해결할 수 있다.
