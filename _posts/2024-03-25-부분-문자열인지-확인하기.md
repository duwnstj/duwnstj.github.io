# 📄문제 풀이 코드
{% highlight java %}
class Solution {
    public int solution(String my_string, String target) {
        int answer = 0;
        if(my_string.contains(target)){
            answer =1;
        }else{
            answer =0;
        }
        return answer;
    }
}
{% endhighlight %}
***
# ⬛문제 풀이
1.대상 문자열에서 찾고자 하는 문자열이 포함되어있는지 여부를 알고 싶을때 사용하는  `contains()`함수를 사용하였고 if조건문을 사용하여 my_string 문자열 안에 target이 포함되어 있다면 `answer=1`을 , else문에 target이 포함되어 있지 않다면 `answer=0`이 저장되게 해서 문제를 풀었다.
