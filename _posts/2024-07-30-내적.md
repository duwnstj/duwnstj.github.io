## 문제 풀이 코드 
```java
class Solution {
    public int solution(int[] a, int[] b) {
        int answer = 0;
        
        for(int i=0;i<a.length;i++){ //1
            answer+= a[i]*b[i]; //2
        }
        return answer;
    }
}
```
## 문제 풀이 설명
- 주석으로 풀이
1. `n`이 a,b 배열의 길이라고 했지만 n은 매개변수에 없었기 때문에 a.length배열 길이를 조건식으로 넣어주었다.
2. answer에 a,b 배열의 각 요소들을 곱해서 더해준다.
