# 문제 풀이 코드
{% highlight java %}
class Solution {
    public int solution(String num_str) {
        int answer = 0;
        for(int i=0;i<num_str.length();i++){
            answer+=Integer.parseInt(String.valueOf(num_str.charAt(i)));
           
        }
        return answer;
    }
}
{% endhighlight %}
***
# ✨문제 풀이
1. 일단 for문을 사용하고 `charAt()`메서드를 사용해서 문자열 하나 하나가 담길수 있게 for문을 통해 answer변수에 담아주고 이때 단일문자를 `String.valueOf()`메서드를 통해 문자열로 바꿔주어야한다. 왜냐하면 여기서 사용한 `Integer.parseInt()`메서드는 문자열을 정수로 바꿔주는 메서드이다. 그렇기 때문에 단일문자를 문자열로 바꿔주어야한다. 이렇게 문제를 해결했다.
