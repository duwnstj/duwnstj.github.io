# ✨문제 풀이 코드
```java
class Solution {
    public int[] solution(int start_num, int end_num) {
        int[] answer = new int[end_num-start_num+1];
        for(int j=0,i=start_num;i<=end_num;i++,j++){
            answer[j]=i;
        }
        return answer;
    }
}
```
***
# 😁문제 풀이
1. answer배열길이를 이렇게 설정하는 이유는 3~10까지라면 10까지 포함이 되어야하기때문에 +1을 해준다.
2. for문으로 `start_num<=end_num`까지 반복문을 돌려준후 그 값을 answer 배열에 차례로 담기게 만들어 주어 문제를 해결했다.
