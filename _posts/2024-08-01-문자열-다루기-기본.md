## 문제 풀이
### 첫번째 풀이(실패)
```java
class Solution {
    public boolean solution(String s) {
       
        if(s.length() !=4 && s.length() != 6){//1
            return false;
        }else{
             for(int i=0;i<s.length();i++){ //2
            if(s.charAt(i)<'0'|| s.charAt(i)>'9'){
                return  false;
            }else{
                return true;
            }
        }
        }
       
        return true;
    }
}
```
### 문제 설명 : 
- **주석으로 풀이 :**
1. 조건 중에 문자열의 길이가 4와 6일 경우에만 true를 반환하라는 조건이 있어서 4와 6이 아닐때 false를 반환하게 해줬다.
2. 문자열의 각 문자가 '0'~'9'사이의 숫자인지 확인한다. 만약 하나라도 숫자가 아니면 'false'를 반환한다.

### 문제점
- `else` 블록에서 `return true`를 사용한것이 문제였다.
- 이렇게 코드를 작성하게 되면 첫번째 문자만 검사한 후 즉시 그 결과를 반환한다. 
- 즉 문자열 전체를 검사하지 않고 결과를 반환하게 되는 것이다.

## 두번째 풀이 (성공)
```java
class Solution {
    public boolean solution(String s) {
       
        if(s.length() !=4 && s.length() != 6){ //1
            return false;
        }
        for(int i=0;i<s.length();i++){ //2
            if(s.charAt(i)<'0'|| s.charAt(i)>'9'){
                return  false;
            }
        }
        return true;
    }
}
```
### 문제 설명:
- 문자열의 길이가 4 또는 6이 아닌 경우 'false'를 반환한다.
- 문자열의 모든 문자가 '0'에서 '9'사이의 숫자인지 확인한 후 , 모든 문자가 숫자일 경우 `true`를 반환한다.
