# 문제 풀이 코드
```java
import java.util.ArrayList;
import java.util.List;
class Solution {
    public List solution(int[] arr) {
        List<Integer> answer = new ArrayList<Integer>();
        for(int i=0;i<arr.length;i++){
            for(int j=0;j<arr[i];j++){
                answer.add(arr[i]);
            }
        }
        
        return answer;
    }
}
```
***
# 문제 풀이
1. ArrayList 클래스 안에 있는 메서드를 사용하기 위해 임포트를 해준다. Java에서 메서드는 기본적으로 원시 타입(primitive type)이나 참조 타입(reference type)을 반환할 수가 있다. 따라서 참조타입인 List 인터페이스를 반환값으로 받아서`public List solution(int[] arr)`이렇게 메서드를 생성해주었다.
2. `List<Integer> answer = new ArrayList<Integer>();` 이렇게 answer의 배열 길이를 선언해주는 이유는 List 인터페이스를 상속받는 ArrayList 객체를 생성해서 동적인 크기를 갖는 배열을 생성해 주기 위해서이다.
List 인터페이스를 사용하여 ArrayList 객체를 참조하면, ArrayList의 기능 뿐만 아니라 List 인터페이스에 정의된 메서드들도 모두 사용할 수 있다. 이는 다형성의 장점 중 하나로, 코드를 더 유연하고 확장 가능하게 만들어준다.
3. `2중 for문`을 사용해서 arr 각 배열의 원소값만큼 반복하는 for문을 만들고 add 메서드를 사용해서 answer배열에 담아준다 이렇게해서 문제를 해결하였다.
