## 1. Vim 소개와 설치 ⚙️

리눅스에서 가장 널리 쓰이는 텍스트 편집기는 **Vi**와 **Vim**입니다.

- **Vi** → 리눅스 기본 내장 편집기 (기능 제한적)
- **Vim (Vi Improved)** → Vi 확장판. **문법 하이라이트, 강력한 편집 기능** 제공. 실무에서 훨씬 많이 사용됨.

👉 따라서 DevOps 환경에서는 대부분 **Vim**을 설치해 사용합니다.

### 📥 CentOS에서 Vim 설치

```bash
sudo yum install vim -y

```text
설치 확인:

```bash
vim --version

```text
출력 예시:

```bash
VIM - Vi IMproved 8.2 (2019 Dec 12, compiled ...)

```text
### 🌐 네트워크 문제 해결

- 인터넷이 안 될 경우 **yum 저장소** 설정을 확인하세요.
- 먼저 네트워크 체크:
    
    ```bash
    ping 8.8.8.8
    ping google.com
    
    ```

## 2. Vim 기본 모드 이해 🎛️

Vim은 **모드 기반 편집기**입니다. 모드를 이해해야 제대로 쓸 수 있습니다.

- 🖥️ **명령 모드 (Command Mode)** → 저장, 종료, 복사, 삭제 등 실행
- ✍️ **입력 모드 (Insert Mode)** → 실제 텍스트 입력 가능 (`i`, `o`)
- ⚡ **확장 모드 (Extended Mode)** → `:` 입력 후 명령 (`:w`, `:q`, `:set nu`)

👉 핵심: **Esc → 명령 모드**, `i`/`o` → 입력 모드.

---

## 3. 파일 생성과 저장·종료 💾

- 새 파일 열기:
    
    ```bash
    vim hello.txt
    
    ```
    
- 저장: `:w`
- 종료: `:q`
- 저장 후 종료: `:wq`
- 저장 안 하고 강제 종료: `:q!`

👉 자주 쓰는 조합은 `:wq`와 `:q!`.

---

## 4. 편집 모드 전환 방법 ✍️

- `i` → 현재 위치에서 입력 시작
- `o` → 다음 줄에 새로 입력 시작
- `Esc` → 입력 모드 종료, 명령 모드로 돌아감

👉 `Esc`는 Vim의 탈출구!

---

## 5. 복사·붙여넣기 📑

- 한 줄 복사: `yy`
- 여러 줄 복사: `nyy` (예: `5yy` → 5줄 복사)
- 붙여넣기:
    - `p` → 현재 줄 아래
    - `P` → 현재 줄 위

👉 `yy` → `p` 조합은 필수 암기!

---

## 6. 잘라내기·삭제 ✂️

- 한 줄 삭제: `dd`
- 여러 줄 삭제: `ndd` (예: `3dd` → 3줄 삭제)
- 잘라낸 줄은 버퍼에 저장 → `p`로 붙여넣기 가능

👉 `dd`는 삭제이자 잘라내기 기능!

---

## 7. 되돌리기와 재실행 🔄

- 실행 취소(Undo): `u`
- 다시 실행(Redo): `Ctrl + r`

👉 실수해도 걱정 없음. `u` → `Ctrl + r` 로 왔다 갔다 가능.

---

## 8. 줄 번호와 이동 ↕️

- 줄 번호 표시: `:se nu`
- 첫 줄 이동: `gg`
- 마지막 줄 이동: `G`
- 특정 줄 이동: `nG` (예: `10G` → 10번째 줄)

👉 `:se nu`는 Vim 시작할 때 항상 써도 좋음.

---

## 9. 파일 내 검색 🔍

- 검색: `/검색어`
- 다음 결과: `n`
- 이전 결과: `N`

⚠️ 대소문자 구분 있음. 필요하면:

```bash
:set ignorecase

```text
---

## 10. 마무리 및 Vim 활용 팁 💡

- 꼭 알아둘 단축키:
    - 저장: `:w`
    - 강제 종료: `:q!`
    - 복사/삭제/붙여넣기: `yy`, `dd`, `p`
    - 검색: `/검색어`

👉 DevOps 환경에서 Vim은 **서버 설정 파일 수정, 로그 분석**에 반드시 쓰입니다.

처음엔 불편하지만, 익숙해지면 **마우스보다 빠른 편집기**가 됩니다.
