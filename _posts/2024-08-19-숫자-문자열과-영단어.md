## 문제 풀이 코드
```java
class Solution {
    public int solution(String s) {
        int answer = 0;
        // 조건 1 : s가 0으로 시작하는 경우는 없다.
        String[] words = {"zero","one","two","three","four","five","six","seven",
                          "eight","nine"}; // 1 
        for(int i=0;i<words.length;i++){ //2
            if(s.contains(words[i])){ //3 
                s = s.replace(words[i],Integer.toString(i)); //4
            }
            
        }
       answer = Integer.parseInt(s); //5
        return answer;
    }
}
```
## 문제 설명
1. one~nine 까지 있는 words 문자열 배열을 만들어준다.
2. 문자열 s 안에 words 배열의 요소들이 포함되어있는지 확인하기 위해words 배열의 길이만큼 반복문을 돌려준다.
3. `contains()` 메서드를 사용하여 s안에 words의 요소들이 있는지 확인한다. 
4. 포함되어있다면 words[i]를 Integer.toString(i)로 변환 시켜준 문자열을 s 문자열에 저장해준다. 
    - 예를 들어 `words[i]`가 `"zero"라면 "0"`로 변환 해서 s에 저장해준다 생각하면 된다.
5. `Integer.parseInt(s)` 메서드를 사용하면 s문자열을 정수로 변환시켜준다. 변환된 s를 answer에 저장해주고, 반환을 해주면 문제를 해결할 수 있다.
